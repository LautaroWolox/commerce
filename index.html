Plataforma enterprise cloud · on-prem resiliente, escalable, segura con IA aplicada al negocio.

Esta plataforma de e-commerce enterprise combina cloud-native (GCP/GKE/Istio) con on-prem (OCP4/Oracle) para obtener resiliencia, escalabilidad y continuidad sin fricción con el legado.

Arquitectura Checkout Kafka OpenAI Blue/Green + Canary Observabilidad Datos Animación E2E

Frontend

  • Angular (Web): SPA con routing, lazy loading e interceptor JWT. Catálogo y búsqueda como observables (RxJS) para UX fluida.
  • Flutter (Mobile): App híbrida con interceptor para Bearer token. Lógica semántica de endpoints, smoke tests de compra/checkout.

Backoffice y Datos

  • Oracle: integra procesos financieros/logísticos. ERP en bounded contexts.
  • PostgreSQL: operaciones transaccionales (órdenes, usuarios, pagos) con consistencia ACID.
  • MongoDB: catálogo flexible, variantes, escalabilidad horizontal.
  • Redis: caché de lecturas y sesiones para baja latencia.

Integración y Procesos

  • Kafka: backbone de eventos versionados con Schema Registry.
  • Camunda BPMN: checkout orquesta reserveStock → authorizePayment → createOrder.
  • OAuth2/OIDC: autenticación estándar, API-Gateway válida JWT y propaga identidades.

Observabilidad y SLOs

  • OpenTelemetry + Dynatrace: métricas y logs con SLOs de negocio (p95/p99).

Entrega Continua

  • Tekton + quality gates: despliegues Canary/Blue-Green controlados por métricas.
